# React / The Pizza Task 

## Getting Started

```bash
# clone repo
git clone https://github.com/zilohumberto/pizza-task-frontend-react front

# navigate to repo
cd front

# install deps
npm install
```

## Build on server
```bash

npm install -g serve
serve -s build
serve -s build -l 3000 > output.log &

```
